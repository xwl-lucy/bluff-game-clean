按钮资源包（1200 DPI）
====================

来源：根据你提供的按键设计图逐个裁切导出
格式：PNG（透明背景）
DPI：1200 x 1200

目录说明：
- section1_large: 第一部分大型主按钮（开始游戏 / 重新开始）
- section2_medium: 第二部分中型按钮（出牌 / 质疑 / 跳过）
- section3_icons: 第三部分小型图标按钮（设置 / 返回 / 对勾 / 关闭）
- section4_tabs: 第四部分选项卡按钮（牌堆 / 手牌 / 弃牌堆 / 记录）
- section5_disabled_example: 第五部分禁用状态示例
- section6_large_compare: 第六部分大型按钮状态对照（设置）
- section7_medium_compare: 第七部分中型按钮状态对照（确认）
- section8_icon_compare: 第八部分小图标按钮状态对照（返回）

命名规则：
- 例如：section2_medium/play_default.png 表示“中型按钮-出牌-默认态”
